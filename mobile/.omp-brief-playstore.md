**`android/app/build.gradle.kts`** — tambahkan signing config untuk release:
- Tambahkan `signingConfigs` block dengan `release` dari keystore
- Gunakan **debug keystore sementara** (untuk dev build — user akan pakai keystore sendiri nanti):
```kotlin
signingConfigs {
    create("release") {
        storeFile = file("debug.keystore")
        storePassword = "android"
        keyAlias = "androiddebugkey"
        keyPassword = "android"
    }
}
```
- Ubah `buildTypes.release.signingConfig` ke `signingConfigs.getByName("release")`
